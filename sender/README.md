# Telegram Bot для отправки HTML писем

Бот для отправки HTML текста на email адреса через Telegram.

## 🚀 Установка

1. Клонируйте репозиторий или скачайте файлы

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Заполните `.env` файл своими данными:
   - `TELEGRAM_BOT_TOKEN` - получите у [@BotFather](https://t.me/BotFather) в Telegram
   - `EMAIL_USER` - ваш email для отправки
   - `EMAIL_PASSWORD` - пароль приложения (для Gmail используйте App Password)
   - `DEFAULT_RECIPIENT` - дефолтный email получателя (опционально)

## 📧 Настройка Gmail

Для использования Gmail необходимо создать пароль приложения:

1. Перейдите в [Настройки аккаунта Google](https://myaccount.google.com/)
2. Включите двухфакторную аутентификацию
3. Перейдите в раздел "Пароли приложений"
4. Создайте новый пароль приложения для "Почта"
5. Используйте этот пароль в `EMAIL_PASSWORD`

## 🏃 Запуск

```bash
python main.py
```

## 📖 Использование

1. Найдите вашего бота в Telegram
2. Отправьте команду `/start`
3. Отправьте команду `/send` для начала отправки письма
4. Следуйте инструкциям бота:
   - Введите email получателя (или используйте `/skip` для дефолтного)
   - Введите тему письма
   - Отправьте HTML текст письма

## 📝 Пример HTML

```html
<html>
<body>
<h1>Привет!</h1>
<p>Это <b>HTML</b> письмо с форматированием.</p>
<ul>
<li>Пункт 1</li>
<li>Пункт 2</li>
</ul>
</body>
</html>
```

## 📁 Структура проекта

```
sender/
├── main.py              # Главный файл запуска бота
├── config.py            # Конфигурация и настройки
├── requirements.txt     # Зависимости проекта
├── .env.example         # Пример файла с переменными окружения
├── README.md           # Документация
├── handlers/           # Обработчики команд и сообщений
│   ├── __init__.py
│   ├── states.py       # FSM состояния для процесса отправки
│   ├── commands.py     # Обработчики команд (/start, /help, /send)
│   └── messages.py     # Обработчик текстовых сообщений

    ├── __init__.py
    └── email_sender.py # Класс для отправки email
```

## 🛠 Технологии

- **aiogram 3.12.0** - асинхронная библиотека для работы с Telegram Bot API
- **FSM (Finite State Machine)** - для управления состояниями диалога
- **python-dotenv** - для работы с переменными окружения

## ⚙️ Команды бота

- `/start` - Начать работу с ботом
- `/help` - Показать справку
- `/send` - Начать процесс отправки письма
- `/skip` - Пропустить ввод email и использовать дефолтный

## 🔒 Безопасность

- Никогда не коммитьте файл `.env` в репозиторий
- Используйте пароли приложений для email
- Храните токены в безопасности
